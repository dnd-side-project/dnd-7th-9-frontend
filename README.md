## 서비스 소개
- URL : https://run-us.netlify.app/
- 공부러를 위한 스터디그룹 목표 달성 보조 서비스. 런:US는 스터디 그룹 목표를 달성할 수 있도록 돕는 보조 서비스로, 팀원들끼리 서로 문제집을 생성하고 풀면서 매번 목표를 인식하게 다시금 도와줍니다.
- Keyword : `#서로에게 내는 문제집` `#다함께 목표 달성` `#나의 공부 기록`


## 실행
```
npm run dev
```
